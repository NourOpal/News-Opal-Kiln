ONE FOLDER PER TWEET — naming convention:

references/X/
  baothiento/                  ← folder per tweet (use handle, or handle-shortid)
    screenshot.png             ← required (the tweet preview)
    video.mp4                  ← optional (if the tweet has a video)
    link.txt                   ← optional (paste the full X URL here, one line)
    caption.txt                ← optional (one-line note for what this is showing)

That's it. Any file named screenshot.* renders as the tile image.
Any file named video.* plays on hover.
Folder name shows as the label on the card.
